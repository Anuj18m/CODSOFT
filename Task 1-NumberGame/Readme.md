# Number Guessing Game (Java)

## 📌 Description

This is a simple Java console-based game where the user tries to guess a randomly generated number between 1 and 100.

## 🚀 Features

* Random number generation
* User input handling
* Limited attempts (5 tries)
* Feedback: Too high / Too low
* Win/Lose condition

## 🛠️ Technologies Used

* Java
* Scanner (for input)
* Random (for number generation)

## ▶️ How to Run

1. Compile:
   javac NumberGame.java

2. Run:
   java NumberGame

## 📷 Output Example

User is prompted to guess a number and receives feedback until correct or attempts are over.
