# ATM System (Java)

## 📌 Description

This project simulates a basic ATM system using Object-Oriented Programming (OOP) concepts in Java.

## 🚀 Features

* Check account balance
* Deposit money
* Withdraw money
* View last transaction
* Input validation to prevent errors

## 🧠 Concepts Used

* Classes and Objects
* Encapsulation
* Method interaction
* Input validation

## 🛠️ Technologies

* Java
* Scanner (for input)

## ▶️ How to Run

1. Compile:
   javac *.java

2. Run:
   java ATM

## 📷 Output

Console-based ATM system with menu-driven interface.
