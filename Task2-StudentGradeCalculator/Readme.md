# Student Grade Calculator (Java)

## 📌 Description

This is a Java console-based application that calculates total marks, average percentage, and assigns grades based on user input.

## 🚀 Features

* Takes marks for multiple subjects
* Calculates total and average
* Finds highest and lowest marks
* Assigns grade (A–F)
* Provides performance description
* Input validation included

## 🛠️ Technologies Used

* Java
* Scanner
* Arrays

## ▶️ How to Run

1. Compile:
   javac StudentGradeCalculator.java

2. Run:
   java StudentGradeCalculator

## 📷 Output

Displays total marks, average percentage, highest & lowest marks, grade, and performance.
